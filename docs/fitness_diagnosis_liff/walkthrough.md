# 修正内容の確認: ボディタイプ診断 LIFF

仕様書に基づき、`index.html` を全面的に刷新しました。女性に寄り添ったプレミアムなデザインと、正確な診断ロジックを統合しています。

## 実施した変更

### 1. ビジュアルデザインの刷新
- **テーマ**: 繊細で華やかな「シャンパンピンク」と「ホットピンク」のグラデーション。
- **UI**: 透過カード（ガラスモーフィズム）と浮遊アニメーションを採用し、女性に好まれる「プレミアム感」を演出。
- **フォント**: `Jost` と `Inter` を使用し、モダンで読みやすいタイポグラフィを実現。

### 2. 診断ロジックの実装
- **スコアリング**: Q2, Q3, Q4 の回答に基づき、合計3点以上で「パーソナルトレーニング」に誘導。
- **タイプ分岐**: 合計2点以下の場合は、Q1 の回答内容に応じて 3つのスタジオタイプ（ボディシェイプ/ファンクショナル/マインドボディ）に自動分岐。
- **データ連携**: 
  - `liff.sendMessages` を用いて、ユーザー自身のトーク画面に診断結果を投稿。
  - `fetch` を用いて GAS サーバー（スプレッドシート）へ回答データを即時送信。
